- Untuk memulai coding harus menggunakan: public/private (sesuai kebutuhan) class namaFile (yaitu nama dari file yang digunakan untuk menulis code)
- Lalu code yang digunakan untuk membuat tulisan muncul adalah print/printf/println (sesuai kebutuhan)
- "print" dapat membuat text yang diinginkan muncul tetapi jika penambahan text digunakan kembali maka text tersebut akan muncul di baris yang sama
- contoh penggunaan print:
System.out.print("hati-hati");
System.out.print("dijalan");
maka yang keluar akan menjadi:  hati-hatidijalan
- "println" dapat membuat text yang diinginkan muncul tetapi jika penambahan text digunakan kembali maka text tersebut akan muncul di baris selanjutnya
- contoh penggunaan println :
System.out.println("hati-hati");
System.out.println("dijalan");
maka yang keluar akan menjadi : hati-hati
                                dijalan
